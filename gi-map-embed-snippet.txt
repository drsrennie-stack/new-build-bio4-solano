GUIDED GI MAP — EMBED (iframe) SNIPPET
=======================================

Paste this into a Canvas page (use the HTML/<> editor), your website, or any LMS
page to embed the Guided GI Map inline.

--- Standard embed (inline on the page) ---

<p style="text-align:center">
  <iframe src="https://drsrennie-stack.github.io/new-build-bio4-solano/guided-gi-map.html"
          width="100%" height="900"
          style="border:1px solid #ECE7DE; border-radius:12px; max-width:1180px"
          title="Guided GI Map"
          allowfullscreen
          loading="lazy"></iframe>
</p>

--- Full-bleed embed (fills the frame, nothing else showing) ---

<iframe src="https://drsrennie-stack.github.io/new-build-bio4-solano/guided-gi-map.html"
        style="position:fixed; inset:0; width:100%; height:100%; border:0"
        title="Guided GI Map"
        allowfullscreen></iframe>

NOTES
-----
• In Canvas: open a Page → Edit → click the "</>" (HTML) button → paste. If Canvas
  strips the iframe, your institution may need the domain (medmasterscollaborative.com)
  added to the allowed-iframes list, OR use the "External URL" / redirect module item
  instead so it opens full-screen.
• For a true "nothing else showing" full-screen experience, prefer a direct link that
  opens in a new tab (target="_blank") over an iframe — an iframe always sits inside
  the host page's chrome.
• The height (900) is a starting point; increase it if the lab feels cramped, or use the
  full-bleed version above for edge-to-edge.
