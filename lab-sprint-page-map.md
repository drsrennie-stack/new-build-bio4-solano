# Lab Sprint Page Map

Each `week-N-lab-sprints.html` master page has a grid of topic cards. Every card links to a per-topic lab sprint page by the exact filename below. For a card to work, a file with that exact name must sit in the `new-build-bio4-solano` repo.

Status as of 2026-05-25: the 8 master pages exist and are wired to the week pages. None of the 32 per-topic files below are in the repo yet, so every topic card currently 404s.

To hook them up: name each lab sprint HTML file exactly as listed and push it to the repo. Or drop your files into the BIOL OO4 Content folder and they can be renamed to match.
